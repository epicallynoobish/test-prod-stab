      Описание

      Переведено с английского языка.-Cucumber - это программный инструмент, используемый программистами, который поддерживает разработку на основе поведения. Центральным в подходе BDD к Cucumber является его синтаксический анализатор простых языков под названием Gherkin. Википедия (Английский язык)

      Описание

      Cucumber is a software tool used by computer programmers that supports behavior-driven development. Central to the Cucumber BDD approach is its plain language parser called Gherkin. It allows expected software behaviors to be specified in a logical language that customers can understand. Wikipedia
      Разработчики: Aslak Hellesøy, Joseph Wilk, Matt Wynne, Gregory Hnatiuk, Mike Sassak
